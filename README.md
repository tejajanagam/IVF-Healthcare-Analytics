# 🏥 IVF Healthcare Analytics

## 📌 Project Overview

This project analyzes IVF (In Vitro Fertilization) treatment data to identify factors affecting pregnancy success, patient outcomes, and treatment performance using Python, SQL, and Power BI.

---

## 🎯 Objective

- Clean and preprocess IVF healthcare data
- Analyze treatment outcomes and patient characteristics
- Identify patterns influencing IVF success rates
- Build an interactive Power BI dashboard for data visualization

---

## 🛠️ Tools & Technologies

- Python
- SQL
- Power BI
- Microsoft Excel

---

## 📊 Project Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. SQL Data Analysis
5. Dashboard Development in Power BI
6. Business Insights
